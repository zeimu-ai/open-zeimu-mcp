# 福島第一・第二原子力発電所の事故により被害を受けられた方々にお支払する賠償金に関する所得税法上の取扱いについて

| 照会の内容 | 丸1 　照会の趣旨（法令解釈・適用上の疑義の要約及び照会者の求める見解の内容） | [別紙の1のとおり](https://www.nta.go.jp/law/bunshokaito/shotoku/111130/besshi.htm#besshi1) |
| 丸2 　照会に係る取引等の事実関係（取引等関係者の名称、取引等における権利・義務関係等） | [別紙の2のとおり](https://www.nta.go.jp/law/bunshokaito/shotoku/111130/besshi.htm#besshi2) |
| 丸3 　丸2 の事実関係に対して事前照会者の求める見解となることの理由 | [別紙の3のとおり](https://www.nta.go.jp/law/bunshokaito/shotoku/111130/besshi.htm#besshi3) |
| 丸4 　関係する法令条項等 | 所得税法第9条　所得税法施行令第30条、第94条 |
| 丸5 　添付書類 | 賠償金に係る案内書、請求書等一式 |

| 丸6 回答年月日 | 平成23年11月30日 |
| 丸7 回答者 | 国税庁課税部長 |

| 国税庁課税部長8 回答内容 | 標題のことについては、ご照会に係る事実関係を前提とする限り、貴見のとおりで差し支えありません。　ただし、次のことを申し添えます。
(1)　ご照会に係る事実関係が異なる場合又は新たな事実が生じた場合は、この回答内容と異なる課税関係が生ずることがあります。
(2)　この回答内容は国税庁としての見解であり、事前照会者の申告内容等を拘束するものではありません。 |
